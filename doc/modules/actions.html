shuffle (list)
Fisher-Yates method to shuffle a list. It consist on permutations of the objects in a list.

Parameters:

  • list A list to shuffle

Returns:

    Nothing

Usage:

    local a_list = {1,2,3,4,5}
shuffle(a_list)
print(a_list)
-- => {3,2,5,1,4}
