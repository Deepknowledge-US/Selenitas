lines_from (file)
Each line in a file will be added to a table, this function returns the table with all lines in a file.

Parameters:

  • file The path to the file to read

Returns:

    Table of strings, a string per line in the file.

Usage:

    my_table = lines_from(path_to_file)
split (pString, pPattern)
Split a string ("pString") using "pPattern" as splitter. eg: split("hello,world!", ",") -> {"hello" , "world!"}

Parameters:

  • pString The string to split.
  • pPattern String to specify the pattern where we must split the string.

Returns:

    Table of strings, as much as the pattern has splitted

Usage:

    my_table = split('hello, world!', ',')
