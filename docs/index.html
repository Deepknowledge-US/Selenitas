Classes

Agent Base agent class, Mobiles, Cells and Relationals are instances of Agent also.
Cell Cells are agents used to represent the space.
Collection Collections are auxiliar structures to generate subsets of the agents of a Family.
Family Family Class is the basis class from which we build the families of agents.
FamilyCell A Family to hold Cell agents, new elements will be added to the collection as Cell instances.
FamilyMobil A Family to hold Mobil agents, new elements will be added to the collection as Mobil instances.
FamilyRelational A Family to hold Relational agents, new elements will be added to the collection as Relational instances.
