Class Relational

Relationals are agents used to create relations between agents.

Methods

Relational:_init (obj)
When a Relational agent is created, some properties are given to it. There are two properties a Relational have to have: source agent and target agent, this determine the direction of the new link. An error message is returned if one of this field is missing. This class is used by FamilyRelational to create new links betwen agents.

Usage:

    Family_of_relationals:add(
    Relational({
        [source]=ag1,
        [target]=ag2,
        [weight]=3.2
    })
)
Relational:ends ()
Returns the agents related by the link.

Usage:

    local extremes = a_relational:ends()
Relational:target ()
As same as edges in a graph, Relationals have a source and a target. This function returns the target Agent of the Relational.

Usage:

    local target_agent = one_of(Links):target()
Relational:source ()
As same as edges in a graph, Relationals have a source and a target. This function returns the source Agent of the Relational.

Usage:

    local source_agent = my_link:source()
