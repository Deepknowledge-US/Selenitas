Params:_init (obj)
One of the first things we do when creating a new file is to create an instance of this class. This instance (called Config by default) holds some parameters to determine some aspects of the world. This class allows users to create elements in interface

Parameters:

  • obj A table with some parameters of the initial configuration.

Returns:

    A Params instance.

Usage:

    Config = Params({
    ['ticks'] = 200,
    ['xsize'] = 15,
    ['ysize'] = 15
})
