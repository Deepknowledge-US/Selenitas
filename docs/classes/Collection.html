Class Collection

Collections are auxiliar structures to generate subsets of the agents of a Family.

They are instance of main class Family, so a Collection will have all methods of a Family to manipulate agents.

Methods

Collection:_init (family) Collection constructor.
Collection:add (object) This function overwrites the add method of the Family class.


Methods

Collection:_init (family)
Collection constructor. It is usually called by some Family methods. A Collection is a set of agents from a Family.

Parameters:

  • family Is the reference to the Family that has created the Collection and is used to update some methods of the collection. It is also usefull when we need to know the main family of the agents in a collection.

Returns:

    A new instance of Collection.
Collection:add (object)
This function overwrites the add method of the Family class. This add does not increment global number of agents.

Parameters:

  • object Is an instance of Agent which is member of any Family, this agent will be added to the Collection.

Returns:

    nothing

Usage:

    Coll_instance:add(existing_agent)
