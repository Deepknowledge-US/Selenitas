is_family (obj)
Given an object this function does a safe check and returns true if the object is an Instance of Family (Collections are also families).

Parameters:

  • obj An object to check

Usage:

    Agents = FamilyMobil()

is_family('hello') -- => false
is_family( Agents ) -- => true

is_in_list (item, elements)
Given an item and a set of elements or a collection, decide if item is included in the set.

Parameters:

  • item An Object (usually an agent).
  • elements List of elements.

Returns:

    Boolean, true if 'item' is present in 'elements'

Usage:

    local my_check = is_in_list( 2, {1,2,3,4})
print(my_check) -- => true
exists (pred)
Returns true if any agent in the family validates a predicate.

Parameters:

  • pred An anonymous function (boolean predicate).

Returns:

    Boolean: True if there is at least one agent that validates the predicate. It also returns in second place an Agent validating the predicate.

See also:

    Family.exists

Usage:

    local response = exists( A_family, function(x) x.label ~= '' end )
if response then
    print('There is almost an agent with a non empty label')
end
all (family, pred)
Checks if all agents in a family validates a condition.

Parameters:

  • family A family to check.
  • pred Anonymous function (boolean predicate).

Returns:

    Boolean. True if all agents in the family validate the condition.

See also:

    Family.all

Usage:

    all(A_family, function(ag) ag.label == '' end)
is_in (family, agent)
Checks if an agent is member of the family.

same_rgb (object_1, object_2)
A check function to compare colors.

Parameters:

  • object_1 It may be an Agent or a 4 parameters vector (rgba).
  • object_2 It may be an Agent or a 4 parameters vector (rgba).

Returns:

    Boolean, true if both colors have the same rgb components.

Usage:

    local check_color   = same_rgb(an_agent_instance, {1, 0.5, 0.5, 1} )
local check_color_2 = same_rgb(an_agent_instance, another_agent )
local check_color_3 = same_rgb( {1, 0.5, 0.5, 1}, {1, 1, 1, 1} )
same_rgba (object_1, object_2)
A check function to compare colors. This function compares also the alpha channel.

Parameters:
